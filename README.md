# Bank Marketing Analysis 📊
An Exploratory Data Analysis (EDA) project focused on analyzing bank marketing campaign data to identify key customer insights and factors driving term deposit subscriptions.

## 📁 Project Structure
* `Bank Marketing Analysis.ipynb` - Jupyter Notebook containing the data cleaning and EDA.
* `bank_final_cleaned.csv` - The cleaned dataset used for analysis.
* `README.md` - Project documentation.

## 🚀 Analysis Workflow
The analysis in the notebook is broken down into three main sections:
1. **Univariate Analysis:** Analyzing individual variables (e.g., age distribution, job types, subscription rates) to understand their standalone characteristics.
2. **Bivariate Analysis:** Exploring the relationship between two variables (e.g., how age or education level impacts the decision to subscribe to a term deposit).
3. **Multivariate Analysis:** Investigating complex interactions across multiple features simultaneously to uncover hidden patterns in customer behavior.

## 🛠️ Technologies Used
* **Python** 3.x
* **Pandas** & **NumPy** for data manipulation and cleaning
* **Matplotlib** & **Seaborn** for static data visualization
* **Plotly** for interactive data visualizations and graphs
